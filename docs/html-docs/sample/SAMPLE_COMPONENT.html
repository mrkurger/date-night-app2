Sample Component

This is a sample component documentation file that demonstrates the HTML structure and styling for the documentation decentralization project.

Overview

The Sample Component is a reusable UI component that provides a standardized way to display information in the Date Night App. It supports various configurations and can be integrated with other components.

Usage

To use the Sample Component, import it from the shared module and include it in your template: 

import { SampleComponent } from '@app/shared/components/sample';

@Component({
  selector: 'app-feature',
  template: `
    <app-sample
      [title]="'Sample Title'"
      [description]="'Sample Description'"
      (action)="handleAction($event)"
    ></app-sample>
  `
})
export class FeatureComponent {
  handleAction(event: any) {
    console.log('Action triggered:', event);
  }
}

Properties

Name Type Default Description
title string '' The title to display in the component
description string '' The description to display in the component
variant 'default' | 'primary' | 'secondary' 'default' The visual variant of the component
disabled boolean false Whether the component is disabled

Events

Name Type Description
action EventEmitter<any> Emitted when the component's action is triggered
focus EventEmitter<FocusEvent> Emitted when the component receives focus
blur EventEmitter<FocusEvent> Emitted when the component loses focus

Methods

The processDataProcesses the input data and returns a formatted result method is used to transform input data before displaying it. It takes a data object and returns a processed version.

The resetStateResets the component to its initial state method can be called to clear any user input and return the component to its default state.

Examples

Basic Usage

<app-sample
  [title]="'Sample Title'"
  [description]="'Sample Description'"
></app-sample>

With Custom Styling

<app-sample
  [title]="'Custom Sample'"
  [description]="'This sample has custom styling'"
  [variant]="'primary'"
  [class]="'custom-class'"
></app-sample>

With Event Handling

<app-sample
  [title]="'Interactive Sample'"
  [description]="'This sample responds to user interaction'"
  (action)="handleAction($event)"
  (focus)="handleFocus($event)"
  (blur)="handleBlur($event)"
></app-sample>

Accessibility

The Sample Component follows WCAG 2.1 guidelines for accessibility:

  • All interactive elements are keyboard accessible
  • Proper ARIA attributes are used for screen readers
  • Color contrast meets AA standards
  • Focus states are clearly visible

Browser Support

The Sample Component is supported in the following browsers:

  • Chrome (latest 2 versions)
  • Firefox (latest 2 versions)
  • Safari (latest 2 versions)
  • Edge (latest 2 versions)

Known Issues

  • The component may not render correctly in IE11
  • Animation performance may be reduced on older mobile devices
