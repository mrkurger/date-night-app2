Tests Glossary

A glossary of functions, methods, and concepts used in the testing infrastructure.

Helper Functions

mockRequest()Function
Creates a mock Express request object for middleware testing. The function accepts an options object to override default properties.
const req = mockRequest({ params: { id: '123' } }); 
const mockRequest = (options = {}) => {
  const req = {
    body: {},
    params: {},
    query: {},
    headers: {},
    cookies: {},
    path: '/test',
    ...options,
  };
  return req;
};
mockResponse()Function
Creates a mock Express response object for middleware testing. The function returns an object with Jest spy methods for common Express response methods.
const res = mockResponse(); 
const mockResponse = () => {
  const res = {};
  res.status = jest.fn().mockReturnValue(res);
  res.json = jest.fn().mockReturnValue(res);
  res.send = jest.fn().mockReturnValue(res);
  res.cookie = jest.fn().mockReturnValue(res);
  res.clearCookie = jest.fn().mockReturnValue(res);
  res.redirect = jest.fn().mockReturnValue(res);
  res.setHeader = jest.fn().mockReturnValue(res);
  res.locals = {};
  return res;
};
mockNextFunction
A Jest spy function that mocks the Express next middleware function.
const next = mockNext; 
const mockNext = jest.fn();
createTestUser()Function
Creates a test user in the database with default or custom data. The function hashes the password and returns the created user object.
const user = await createTestUser({ email: 'custom@example.com' });
createTestUsers()Function
Creates multiple test users in the database. The function accepts a count and base user data, and returns an array of created user objects.
const users = await createTestUsers(5, { role: 'user' });
generateTestToken()Function
Generates a valid JWT token for testing authentication. The function accepts a user ID and additional claims.
const token = generateTestToken(user._id, { role: 'admin' });
authenticatedRequest()Function
Adds authentication headers to a Supertest request. The function accepts a request object and a JWT token.
const response = await authenticatedRequest(request(app).get('/api/users'), token);
createError()Function
Creates an error object with a status code for testing error handling middleware.
const error = createError('Not found', 404);
createTimingUtil()Function
Creates a utility for measuring execution time in performance tests.
const timer = createTimingUtil(); /* ... */ const elapsed = timer.getElapsedTimeInMs();

Test Data Constants

TEST_USER_DATAConstant
Default test user data used for creating test users.
TEST_AD_DATAConstant
Default test advertisement data used for creating test ads.
TEST_MESSAGE_DATAConstant
Default test message data used for creating test messages.
