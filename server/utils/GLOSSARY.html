Validation System - Glossary

Classes

ValidationUtils

Primary class containing validation utilities and schemas.

File: server/utils/ValidationUtils.ts

Methods

validateWithZod

Validates request data using a Zod schema.

static validateWithZod(schema: z.ZodType, source: 'body' | 'query' | 'params' = 'body')

validate

Validates request data using express-validator chains.

static validate(validations: ValidationChain[])

sanitizeHtml

Sanitizes HTML content to prevent XSS attacks.

static sanitizeHtml(html: string): string

sanitizeEmail

Validates and sanitizes email addresses.

static sanitizeEmail(email: string): string

Schemas

zodSchemas

Common Zod schemas for reuse across the application.

  • objectId: Validates MongoDB ObjectId format
  • email: Validates email addresses
  • password: Validates password complexity
  • coordinates: Validates geographic coordinates
  • pagination: Validates pagination parameters
  • phoneNumber: Validates phone number format

Type Guards

isString

Type guard for checking if a value is a string.

static isString(value: unknown): value is string

isNumber

Type guard for checking if a value is a number.

static isNumber(value: unknown): value is number

Common Validations

commonValidations

Pre-configured validation chains for common use cases.

  • pagination: Validates page and limit parameters
  • objectId: Validates MongoDB ObjectId parameters
  • coordinates: Validates geographic coordinate arrays
