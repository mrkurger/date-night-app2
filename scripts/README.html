Documentation Migration Scripts

This directory contains scripts for migrating Markdown documentation to HTML as part of the documentation decentralization project.

Scripts Overview

  • manual_doc_migration.js - General migration helper
  • markdown_to_html_converter.js - Converts Markdown to HTML
  • create_folder_docs.js - Creates folder documentation structure
  • update_migration_checklist.js - Updates migration checklist
  • tooltip_helper.js - Adds tooltips and links to function references

Usage

The recommended way to use these scripts is through the migrate-docs.sh shell script in the root directory. This script provides a convenient interface to the migration tools.

Creating Folder Documentation

./migrate-docs.sh create-folder-docs <folder-path>

Converting Markdown to HTML

./migrate-docs.sh convert-md-to-html <markdown-file> <output-html-file>

Updating the Migration Checklist

./migrate-docs.sh update-checklist <markdown-file> <html-file> <status>

Auditing Documentation

./migrate-docs.sh audit-docs

Direct Script Usage

If you need to use the scripts directly, you can do so as follows:

manual_doc_migration.js

node manual_doc_migration.js --create-folder-docs <folder-path>
node manual_doc_migration.js --convert-md-to-html <markdown-file> <output-html-file>
node manual_doc_migration.js --update-index

markdown_to_html_converter.js

node markdown_to_html_converter.js <markdown-file> <output-html-file> [--add-tooltips]

create_folder_docs.js

node create_folder_docs.js <folder-path>

update_migration_checklist.js

node update_migration_checklist.js <markdown-file> <html-file> <status>

Templates

The templates directory contains HTML templates used by the migration scripts:

  • doc_template.html - Template for HTML documentation files

Dependencies

These scripts require the following dependencies:

  • marked - For converting Markdown to HTML

You can install the dependencies with:

npm install marked
