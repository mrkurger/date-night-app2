A

A AppCard

Component UI Element Layout

A card component for displaying advertiser information with various layouts and features. This component has been migrated from Emerald UI to use Nebular's card, badge, button, and icon components.

Properties

Name Type Default Description
ad Ad required The advertisement data to display
layout 'tinder' | 'netflix' | 'list' 'netflix' The layout style of the card
showActions boolean true Whether to show action buttons
showDescription boolean true Whether to show the description
isOnline boolean false Whether the advertiser is online

Events

Name Type Description
viewDetails EventEmitter Emitted when the card is clicked
like EventEmitter Emitted when the like button is clicked
chat EventEmitter Emitted when the chat button is clicked
share EventEmitter Emitted when the share button is clicked
swiped EventEmitter<{ direction: 'left' | 'right'; adId: string }> Emitted when the card is swiped (Tinder layout)

Example

<nb-card
  [ad]="adData"
  layout="netflix"
  [showActions]="true"
  [showDescription]="true"
  [isOnline]="true"
  (viewDetails)="onViewDetails($event)"
  (like)="onLike($event)"
  (chat)="onChat($event)"
  (swiped)="onSwiped($event)"
></nb-card>

A Avatar

Component UI Element

A user avatar component that displays a user's image or initials with optional dropdown menu. This component has been migrated from Emerald UI to use Nebular's icon and context menu components.

Properties

Name Type Default Description
imageUrl string '/assets/img/default-profile.jpg' URL for the avatar image
name string '' User name (used for initials if no image)
size 'small' | 'medium' | 'large' 'medium' Size of the avatar
isOnline boolean false Whether to show an online status indicator
showDropdown boolean false Whether to show a dropdown menu
dropdownItems Array [] Items to display in the dropdown menu
useNebularContextMenu boolean true Whether to use Nebular's context menu or custom dropdown

Events

Name Type Description
avatarClick EventEmitter Emitted when the avatar is clicked
itemClick EventEmitter Emitted when a dropdown item is clicked

Example

<nb-user
  imageUrl="https://example.com/avatar.jpg"
  name="John Doe"
  size="medium"
  [isOnline]="true"
  [showDropdown]="true"
  [dropdownItems]="[
    { id: 'profile', label: 'Profile', icon: 'fa-user', route: '/profile' },
    { id: 'settings', label: 'Settings', icon: 'fa-cog', route: '/settings' },
    { divider: true },
    { id: 'logout', label: 'Logout', icon: 'fa-sign-out-alt' }
  ]"
  (avatarClick)="onAvatarClick()"
  (itemClick)="onItemClick($event)"
></nb-user>

P

P PageHeader

Component UI Element Navigation

A page header component that displays a title, subtitle, breadcrumbs, and action buttons. This component has been migrated from Emerald UI to use Nebular's breadcrumbs, buttons, and user components.

Properties

Name Type Default Description
title string '' The title of the page
subtitle string undefined The subtitle of the page
breadcrumbs Array<{label: string, url?: string}> [] Breadcrumb navigation items
actions Array [] Action buttons to display in the header
backLink string undefined URL for the back navigation link
backgroundImage string undefined URL for a background image
avatarUrl string undefined URL for an avatar image
avatarName string undefined Name for the avatar (used for initials if no image)
avatarIsOnline boolean undefined Whether the avatar should show an online status indicator

Events

Name Type Description
actionClick EventEmitter Emitted when an action button is clicked

Example

<nb-header
  title="Dashboard"
  subtitle="Welcome to your dashboard"
  [breadcrumbs]="[
    { label: 'Home', url: '/' },
    { label: 'Dashboard' }
  ]"
  [actions]="[
    { 
      id: 'add', 
      label: 'Add Item', 
      icon: 'fa-plus', 
      variant: 'primary' 
    },
    { 
      id: 'refresh', 
      label: 'Refresh', 
      icon: 'fa-sync', 
      variant: 'secondary' 
    }
  ]"
  backLink="/home"
  (actionClick)="onActionClick($event)"
></nb-header>

L

L Label

Component UI Element

A label/tag component for displaying status, categories, or other metadata. This component has been migrated from Emerald UI to use Nebular's tag component.

Properties

Name Type Default Description
text string '' The text to display in the label
variant 'primary' | 'success' | 'warning' | 'danger' | 'info' | 'neutral' 'primary' The color variant of the label
size 'small' | 'medium' | 'large' 'medium' The size of the label
icon string undefined Optional icon to display before the text
rounded boolean false Whether to use rounded corners
outlined boolean false Whether to use an outlined style
pill boolean false Whether to use a pill shape (fully rounded)

Example

<nb-tag
  text="New"
  variant="success"
  size="medium"
  icon="star-outline"
  [outlined]="true"
  [pill]="true"
></nb-tag>

I

I InfoPanel

Component UI Element Layout

A panel component that displays information in a structured format. This component has been migrated from Emerald UI to use Nebular's card, badge, progress bar, and accordion components.

Properties

Name Type Default Description
title string '' The title of the panel
subtitle string undefined The subtitle of the panel
items Array [] The items to display in the panel
variant 'default' | 'bordered' | 'shadowed' 'default' The visual style of the panel
collapsible boolean false Whether the panel can be collapsed
initiallyCollapsed boolean false Whether the panel starts collapsed

InfoPanelItem Interface

Name Type Description
label string The label for the item
value string | number | boolean The value of the item
icon string Optional icon for the item
type 'text' | 'badge' | 'progress' | 'boolean' The type of value display
variant 'primary' | 'success' | 'warning' | 'danger' | 'info' | 'neutral' The color variant for badges and progress bars

Example

<app-info-panel
  title="User Information"
  subtitle="Personal details"
  [items]="[
    { label: 'Name', value: 'John Doe', icon: 'fa-user' },
    { label: 'Email', value: 'john.doe@example.com', icon: 'fa-envelope' },
    { label: 'Status', value: 'Active', type: 'badge', variant: 'success' },
    { label: 'Profile Completion', value: 75, type: 'progress', variant: 'primary' },
    { label: 'Verified', value: true, type: 'boolean' }
  ]"
  variant="bordered"
  [collapsible]="true"
></app-info-panel>

F

F FloatingActionButton

Component UI Element Navigation

A floating action button (FAB) is a circular button that represents the primary action in an application. It floats above the UI and is used for promoting a primary action. This component has been migrated from Emerald UI to use Nebular's button components and styling.

Properties

Name Type Default Description
color 'primary' | 'secondary' | 'success' | 'danger' | 'warning' | 'info' 'primary' The color theme of the button
icon string 'plus' The icon to display inside the button
size 'small' | 'medium' | 'large' 'medium' The size of the button
position 'bottom-right' | 'bottom-left' | 'top-right' | 'top-left' | 'center' 'bottom-right' The position of the button on the screen
disabled boolean false Whether the button is disabled
label string '' Accessible label for the button
tooltipText string '' Text to display in the tooltip
showTooltip boolean false Whether to show the tooltip
menuItems Array<{id: string, label: string, icon: string}> [] Items to display in the menu when the button is clicked

Events

Name Type Description
click EventEmitter Emitted when the button is clicked
menuItemClick EventEmitter<{id: string, event: MouseEvent}> Emitted when a menu item is clicked

Example

<app-floating-action-button
  color="primary"
  icon="plus"
  size="medium"
  position="bottom-right"
  [menuItems]="[
    { id: 'add', label: 'Add Item', icon: 'plus' },
    { id: 'edit', label: 'Edit', icon: 'edit' }
  ]"
  (click)="onFabClick($event)"
  (menuItemClick)="onMenuItemClick($event)"
></app-floating-action-button>

C

C CardGrid

Component Layout UI Element

A grid layout for displaying cards in a responsive grid. Supports different layouts including standard grid, masonry, and Netflix-style horizontal scrolling. This component has been migrated from Emerald UI to use Nebular's card components and layout system.

Properties

Name Type Default Description
items Array [] The items to display in the grid
layout 'grid' | 'masonry' | 'netflix' 'grid' The layout style of the grid
columns number 3 Number of columns in the grid
gap number 24 Gap between grid items in pixels
loading boolean false Whether the grid is in a loading state
animated boolean true Whether to animate the grid items when they appear
cardLayout 'horizontal' | 'vertical' 'vertical' Layout orientation for the cards

Events

Name Type Description
itemClick EventEmitter Emitted when a grid item is clicked
cardClick EventEmitter Emitted when a card is clicked, with the card ID
actionClick EventEmitter<{id: string, action: string}> Emitted when a card action is clicked

Example

<app-card-grid
  [items]="cardItems"
  layout="grid"
  [columns]="3"
  [gap]="24"
  [loading]="isLoading"
  [animated]="true"
  (itemClick)="onItemClick($event)"
  (cardClick)="onCardClick($event)"
  (actionClick)="onActionClick($event)"
></app-card-grid>

S

S SkeletonLoader

Component UI Element

A skeleton loader component that displays placeholder content while data is being loaded. This component has been migrated from Emerald UI to use Nebular's nb-skeleton component.

Properties

Name Type Default Description
type 'text' | 'circle' | 'rectangle' | 'card' | 'profile' | 'list' 'text' The type of skeleton to display
count number 1 Number of skeleton items to display
animated boolean true Whether to animate the skeleton
width string null Width of the skeleton (CSS value)
height string null Height of the skeleton (CSS value)
borderRadius string null Border radius of the skeleton (CSS value)

Example

<!-- Simple text skeleton -->
<app-skeleton-loader 
  type="text"
  [count]="3"
  [animated]="true"
></app-skeleton-loader>

<!-- Card skeleton -->
<app-skeleton-loader 
  type="card"
  [animated]="true"
></app-skeleton-loader>

<!-- Custom dimensions -->
<app-skeleton-loader 
  type="rectangle"
  width="200px"
  height="100px"
  borderRadius="8px"
  [animated]="true"
></app-skeleton-loader>

T

T Toggle

Component UI Element Form Control

A toggle switch component that allows users to switch between two states. This component has been migrated from Emerald UI to use Nebular's nb-toggle component.

Properties

Name Type Default Description
label string undefined Label text for the toggle
labelPosition 'left' | 'right' 'right' Position of the label relative to the toggle
color 'primary' | 'success' | 'warning' | 'danger' | 'info' 'primary' Color of the toggle when checked
disabled boolean false Whether the toggle is disabled
name string undefined Name attribute for the toggle input
required boolean false Whether the toggle is required
ariaLabel string undefined ARIA label for accessibility
value boolean false Current value of the toggle

Events

Name Type Description
change EventEmitter Emitted when the toggle value changes

Example

<app-toggle
  label="Enable notifications"
  labelPosition="right"
  color="primary"
  [disabled]="false"
  [required]="true"
  [(ngModel)]="notificationsEnabled"
  (change)="onNotificationsToggle($event)"
></app-toggle>

A toggle switch component that can be used as a boolean input control. This component has been migrated from Emerald UI to use Nebular's nb-toggle component.

Properties

Name Type Default Description
id string auto-generated The ID of the toggle input
name string '' The name of the toggle input
value boolean false The current value of the toggle
disabled boolean false Whether the toggle is disabled
required boolean false Whether the toggle is required
color 'primary' | 'success' | 'warning' | 'danger' | 'info' 'primary' The color theme of the toggle
size 'small' | 'medium' | 'large' 'medium' The size of the toggle
label string '' The label text for the toggle
labelPosition 'left' | 'right' 'right' The position of the label relative to the toggle
ariaLabel string '' Accessible label for the toggle

Events

Name Type Description
valueChange EventEmitter Emitted when the toggle value changes

Example

<app-toggle
  label="Enable notifications"
  labelPosition="right"
  color="primary"
  size="medium"
  [value]="notificationsEnabled"
  [disabled]="false"
  [required]="true"
  (valueChange)="onNotificationsToggle($event)"
></app-toggle>
