Emerald UI Components Changelog

Version 2.0.0

June 10, 2024

Major Changes

  • BREAKING Migrated from custom Emerald UI components to Nebular UI framework components
  • REFACTOR Updated all component styling to use Nebular theme variables instead of custom CSS variables

Component Updates

  • REFACTOR FloatingActionButton: Replaced custom implementation with Nebular button components and styling
  • REFACTOR CardGrid: Updated to use Nebular card components and layout system
  • REFACTOR Toggle: Replaced custom toggle implementation with Nebular's nb-toggle component
  • REFACTOR SkeletonLoader: Replaced custom skeleton implementation with Nebular's nb-skeleton component
  • REFACTOR PageHeader: Updated to use Nebular's breadcrumbs, buttons, and user components
  • REFACTOR Avatar: Updated to use Nebular's icon and context menu components
  • REFACTOR Toggle: Replaced custom toggle implementation with Nebular's nb-toggle component
  • REFACTOR InfoPanel: Updated to use Nebular's card, badge, progress bar, and accordion components
  • REFACTOR AppCard: Replaced custom implementation with Nebular's card, badge, button, and icon components
  • REFACTOR Label: Replaced custom implementation with Nebular's tag component

Style Updates

  • FEATURE Added support for Nebular's theming system, enabling better dark mode support
  • FIX Improved accessibility by using Nebular's built-in accessibility features
  • REFACTOR Standardized component naming from emerald-* to nb-* prefix

Migration Guide

When using these components, please note the following changes:

  • Component selectors remain the same for backward compatibility
  • CSS classes have changed from emerald-* to nb-*
  • Some prop names may have changed to match Nebular's API
  • Theme customization should now be done through Nebular's theming system
