Angular Client Component Glossary

Nebular Components

The following components from the Nebular UI library are used throughout the application:

NbLayout

Layout Core

The main layout container for the application. Used as the root component for the app structure.

Usage: <nb-layout></nb-layout>

NbCard

Container Data Display

A content container component for organizing information into sections with header, body, and footer.

Usage: <nb-card></nb-card>

NbButton

Control Interactive

A button component with various styles, sizes, and states.

Usage: <button nbButton></button>

NbFormField

Form Control

A wrapper for form controls with integrated validation display.

Usage: <nb-form-field></nb-form-field>

Custom Components

The following are custom components that could potentially be replaced with Nebular equivalents:

CardComponent

Custom Container

A custom card component that adds styling options beyond Nebular's card. Could be replaced with nb-card.

Usage: <app-card></app-card>

SkeletonLoaderComponent

Custom Placeholder

A custom skeleton loading component. Could be replaced with Nebular's nb-skeleton.

Usage: <app-skeleton></app-skeleton>

ButtonComponent

Custom Control

A custom button wrapper around Nebular's button. Consider using nb-button directly.

Usage: <app-button></app-button>

CarouselComponent

Custom Data Display

A custom carousel component as Nebular doesn't provide one. Keep but ensure styling consistency.

Usage: <app-carousel></app-carousel>
