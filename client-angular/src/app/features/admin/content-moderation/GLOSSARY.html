Components

ContentModerationComponent

Component

Main component for content moderation by administrators. Allows reviewing and moderating user-submitted media content.

Features include filtering, sorting, pagination, and moderation actions (approve/reject) for media items.

Source: /features/admin/content-moderation/content-moderation.component.ts

ModerationModalComponent

Component

Component for displaying and handling media moderation in a modal dialog. Provides media preview and moderation form.

Source: /features/admin/content-moderation/moderation-modal/moderation-modal.component.ts

Interfaces

PendingMedia

Interface

Represents a media item pending moderation.

interface PendingMedia {
  _id: string;
  adId: string;
  adTitle: string;
  url: string;
  type: 'image' | 'video';
  createdAt: Date;
  pendingSince: Date;
}

Source: /core/models/media.interface.ts

ModerationRequest

Interface

Represents a request to moderate content with decision and notes.

interface ModerationRequest {
  status: 'approved' | 'rejected';
  notes: string;
}

Source: /core/models/media.interface.ts
