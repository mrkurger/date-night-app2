Glossary: Advertiser browsing alternate

This file contains a glossary of functions, methods, and components defined in the Advertiser browsing alternate component/module.

AdvertiserBrowsingAlternateComponent

Main component for the alternate advertiser browsing experience. Provides a modern interface for browsing advertisers with multiple view modes and interaction options.

Properties

isLoggedIn: boolean
Tracks user authentication state
isAdvertiser: boolean
Indicates if current user has advertiser privileges
searchQuery: string
Current search input value
activeView: ViewMode
Currently selected view layout ('grid' | 'cards' | 'map' | 'nearby' | 'featured')
displayedAdvertisers: Advertiser[]
List of advertisers currently shown in the view
premiumAds: Advertiser[]
List of premium advertisers for featured display

Methods

handleSearch(event?: Event): void
Processes search form submissions and updates results
onViewChange(event: { value: string }): void
Handles switching between different view layouts
onAdvertiserLike(advertiser: Advertiser): void
Processes user likes/favorites for advertisers
onAdvertiserChat(advertiser: Advertiser): void
Initiates chat sessions with advertisers
onAdvertiserProfile(advertiser: Advertiser): void
Navigates to detailed advertiser profiles
updateUserMenus(): void
Updates navigation menus based on user state

Interfaces

Advertiser

id: number
Unique identifier
name: string
Display name
age: number
Advertiser age
location: string
Geographic location
description: string
Profile description
tags: string[]
Categorization tags
isPremium: boolean
Premium status flag
isOnline: boolean
Current online status

Child Components

AltNetflixViewComponent

Grid-based layout for browsing advertisers

AltTinderViewComponent

Card-swipe interface for browsing advertisers

AltPremiumAdsSectionComponent

Displays featured premium advertisers

AltPaidPlacementSidebarComponent

Sidebar component for paid advertisement placements

AltChatWidgetComponent

Floating chat widget for user communication

PrimeNG Components Used

p-panel
Main layout container component
p-toolbar
Navigation and action bar component
p-button
Interactive button component
p-menu, p-tieredMenu
Navigation menu components
p-tabView
View switching component
p-inputText
Text input component
