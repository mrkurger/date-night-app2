Changelog: Icon

This file tracks changes to the Icon component/module.

Version 1.0.0 - 2024

Breaking Changes

  • Migrated from Font Awesome and Eva Icons to PrimeIcons
  • Removed @fortawesome dependencies
  • Removed Eva Icons dependencies
  • Updated component to use PrimeIcons classes

Added

  • New icon mapping system in IconService
  • Support for PrimeIcons
  • Improved size and color customization
  • Legacy icon name support through mappers

Changed

  • Updated component template to use PrimeIcons classes
  • Simplified component styles
  • Enhanced icon name conversion logic
  • Improved documentation and type definitions

Removed

  • Font Awesome dependencies and integrations
  • Eva Icons dependencies and configurations
  • Legacy icon pack registrations

How to use this file

When making changes to this component, add an entry to the "Unreleased" section above. When a release is made, create a new version section and move the unreleased entries there.
