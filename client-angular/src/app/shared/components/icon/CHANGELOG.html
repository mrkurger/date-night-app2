Changelog: Icon

This file tracks changes to the Icon component/module.

Version 2.0.0 - May 25, 2025

Breaking Changes

  • Completed migration from Eva Icons to PrimeIcons
  • Removed all Eva Icons dependencies
  • Updated all icon references in client-angular
  • Consolidated icon styles to use only PrimeIcons

Added

  • Comprehensive Eva Icons to PrimeIcons mapping in IconService
  • Support for PrimeIcons animation classes
  • Improved icon state management
  • Enhanced accessibility for icon-only buttons

Changed

  • Updated all icon template references to use pi- prefix
  • Enhanced icon size scaling system
  • Improved icon color inheritance
  • Updated documentation with PrimeIcons guidelines

Removed

  • Eva Icons dependencies and CDN references
  • Legacy Eva Icons style overrides
  • Eva Icons related configuration

Version 1.0.0 - April 1, 2025

Initial Release

  • Initial implementation of IconService
  • Basic support for Eva Icons
  • Icon component with customization options
  • Core icon mapping functionality

How to use this file

When making changes to this component, add an entry to the "Unreleased" section above. When a release is made, create a new version section and move the unreleased entries there.
