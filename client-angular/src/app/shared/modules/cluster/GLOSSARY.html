Cluster Module - Glossary

Classes

ClusterModule
NgModule
A module that provides functionality for clustering map markers to improve performance and usability when displaying many markers on a map.
import { ClusterModule } from '../../shared/modules/cluster/cluster.module'; @Component({ // ... imports: [ ClusterModule, // other imports ] }) export class MapComponent { }

Interfaces

MarkerClusterConfig
Interface
Configuration options for marker clustering behavior.
const clusterConfig: MarkerClusterConfig = { maxZoom: 18, gridSize: 60, minimumClusterSize: 3 };

Constants

DEFAULT_CLUSTER_OPTIONS
Constant
Default configuration options for marker clustering.
// Default options that will be used if not overridden const DEFAULT_CLUSTER_OPTIONS = { maxZoom: 16, gridSize: 50, minimumClusterSize: 2 };
