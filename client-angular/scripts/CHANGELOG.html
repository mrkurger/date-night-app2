Overview

This document tracks all notable changes to the Angular client fix scripts. Each version includes details about new features, bug fixes, and improvements.

Version 1.0.0 May 24, 2024

Initial release of the Angular client fix scripts.

Added

  • New: Created fix-all-issues.cjs script to run all fixes in sequence
  • New: Created fix-css-paths.cjs script to create placeholder CSS files
  • New: Created fix-index-html.cjs script to update index.html with Eva Icons and CSP
  • New: Created fix-styles.cjs script to add CSS variables for Nebular theme
  • New: Created fix-nebular-imports.cjs script to consolidate Nebular imports
  • New: Created fix-remaining-nebular-issues.cjs script to fix component formatting
  • New: Created fix-chat-message-interface.cjs script to update the ChatMessage interface
  • New: Added comprehensive README.md with usage instructions

Fixed

  • Fix: Eva Icons CSS loading issues by adding CDN link in index.html
  • Fix: Nebular module imports in component files
  • Fix: Component imports array formatting in standalone components
  • Fix: Added missing CSS variables for Nebular theme in styles.scss
  • Fix: Updated Content Security Policy to allow loading from CDNs
  • Fix: Updated ChatMessage interface with missing properties
  • Fix: Fixed issues with NbTagComponent and separatorKeys
  • Fix: Fixed dialog reference issues in components

Improved

  • Improved: Script execution with detailed logging
  • Improved: Error handling in all scripts
  • Improved: Documentation with usage examples and troubleshooting tips
